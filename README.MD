# GM PDF Viewer

How to open:

1. Download and Install Node.js v20+
2. For Windows users, open termin in admin privileges, and type ```Set-ExecutionPolicy Unsigned```
3. Open terminal in both pdf-tool-backend and pdf-tool-frontend
4. Type ```npm i``` in both the terminals
5. once completed type ```npm start``` in both the terminals.
6. If there was no default browser set, click on the link ```http://localhost:4200``` to view on browser
